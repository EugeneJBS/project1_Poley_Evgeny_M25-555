# Лабиринт сокровищ

## Описание проекта
Это консольная текстовая игра, реализованная на чистом Python. Вы — исследователь древнего лабиринта. Вам необходимо перемещаться по комнатам, собирать артефакты, разгадывать загадки и найти легендарное сокровище, скрытое в охраняемом сундуке.

Проект демонстрирует навыки процедурного программирования, работы со структурами данных, использования математических функций для создания псевдослучайных событий и управления проектом через **Poetry**.

### Основные возможности
- **Исследование:** Система комнат с уникальными описаниями и взаимосвязанными переходами.
- **Инвентарь:** Возможность подбирать, хранить и использовать предметы.
- **Загадки:** Интерактивные головоломки, блокирующие доступ к наградам или новым локациям.
- **Случайные события:** Реализация системы ловушек и находок на основе математических алгоритмов (без использования модуля `random`).
- **Качество кода:** Код полностью соответствует стандарту PEP8 (проверено линтером Ruff).
## Требования
* Python версии **3.11** или выше.
* Установленный пакетный менеджер **Poetry**.

## Игровое управление

### Перемещение
- `go north` / `north` — идти на север.
- `go south` / `south` — идти на юг.
- `go east` / `east` — идти на восток.
- `go west` / `west` — идти на запад.

### Взаимодействие
- `look` — осмотреть текущую комнату (описание, выходы и предметы).
- `take <предмет>` — поднять предмет (например, `take torch`).
- `use <предмет>` — использовать предмет из инвентаря (например, `use bronze_box`).
- `solve` — попытаться решить загадку в комнате или открыть финальный сундук.
- `inventory` — показать список собранных предметов.

### Общие команды
- `help` — вывести список всех команд.
- `exit` — завершить игру и выйти.

## Команды разработки (Makefile)
- `make install` — установка проекта и зависимостей.
- `make project` — запуск игры.
- `make lint` — проверка кода линтером Ruff.
- `make build` — сборка пакета.
- `make package-install` — установка собранного `.whl` пакета в систему.

## Технические детали
- **Методика разработки:** Github Flow.
- **Библиотеки:** Используется только стандартная библиотека Python (`math`).
- **Архитектура:**
    - `main.py` — игровой цикл и обработка ввода.
    - `constants.py` — конфигурация игрового мира (комнаты, команды).
    - `player_actions.py` — логика действий игрока и изменение состояния.
    - `utils.py` — вспомогательные инструменты и расчет случайных событий.
